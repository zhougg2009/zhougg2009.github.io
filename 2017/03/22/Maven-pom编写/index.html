Maven pom.xml编写

Mar 22, 2017 | Maven

pom.xml

pom(Project Object Model，项目对象模型)定义了项目的基本信息，用于描述项目如何构建，声明项目依赖等等。

最简单的pom示例：

  • modelVersion指定了当前的POM模型版本，maven2和maven3固定为4.0.0
  • groupId定义了所属项目组，往往和组织相关，比如：com.baidu.appname
  • artifactId定义了当前项目组中的唯一Id，比如：appname-web，appname-service，appname-dao
  • version定义了当前项目开发的当前版本，SNAPSHOT表示快照，还在开发中，不稳定
  • package定义打包的类型，jar，war，pom
  • name定义项目的友好名称，不是必须的
  • describe定义了项目的描述，不是必须的
